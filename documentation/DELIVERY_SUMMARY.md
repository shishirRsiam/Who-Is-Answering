## 💡 Pro Tips

### Scale to Multiple Containers
```bash
docker-compose up -d --scale backend=5
```

### View Logs
```bash
docker-compose logs -f backend
```

### Reset Everything
```bash
docker-compose down -v
docker-compose up -d --build
```

### Database Shell
```bash
docker-compose exec db psql -U scalinguser -d scalingdb
```
